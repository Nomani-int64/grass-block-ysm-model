# grass-block-ysm-model

这是一个以Minecraft方块为原型的YSM模型。

A simple YSM model of cute Minecraft blocks.

## 等下，YSM模型是啥？ But wait, what is a YSM model?

这个模型需要配合Minecraft Java Edition的“是，史蒂夫模型”（Yes Steve Model）模组使用。YSM允许你在游戏中使用自定义的玩家模型、材质和动画。

This model should be used via the **Yes Steve Model mod** of Minecraft Java Edition. YSM mod enables you to use custom models, textures and animations in game.

## 如何使用？ How to use it?

如果只是想使用这个模型，您需要先在游戏中安装YSM模组，并将模型的文件夹或ZIP压缩包放到游戏的`config/yes_steve_model/custom`目录下。然后您就可以在游戏内的菜单里找到它。

有关模组的具体使用方法，包括模型的修改等，请参见模组[官方页面](https://ysm.cfpa.team/)。

If you simply want to equip yourself with the model, you need to install YSM mod in your game first, then place the model folder or ZIP file in the `config/yes_steve_model/custom` folder. Then you should be able to see it in the in-game menu.

About the further usage of the mod, including the modification of the model, please refer to its [official page.](https://ysm.cfpa.team/)
